Detailed Description

Used for generating Machine objects.

The MachineFactory class contains functionality to accept machine information and create a Machine object. The first time that the class is called it will create a new type of Machine object. When it is subsequently called, it will use a prototype of the machine type to create a new object by cloning it.

Machine object generation method.

This method generates Machine objects given a machine's identification number, name, and type.

Parameters
id- a constant reference to the identification number of the machine
name- a constant reference to the name of the machine
type- a constant reference to teh type of the machine
Returns
a pointer to a Machine object

If the prototype for the type points to an object, it clones it and sets the resulting clone's name and id to the input parameters. If the prototype is set to nullptr, then this method creates a new object of the specified type and assigns it to the prototype. The method returns the new object.
